University Ranking Logic:
	Iterate through list, giving each one a score based on the following:
		League:
			Bronze     - 1000
			Silver     - 2000
			Gold       - 3000
			Platinum   - 4000
			Diamond    - 5000
			Masters    - 10000
			Challenger - 20000
		Division:
			V   - 0
			IV  - 200
			III - 400
			II  - 600
			I   - 800
		LP:
			Simply add the LP on!

	Examples:
		Platinum 4, 69LP is:
			4269 pts
		Gold 1, 0LP is:
			3800 pts
		Challenger 429LP is:
			20429 pts
		Bronze 2, 100LP is:
			1700 pts
		Silver 5, 12LP is:
			2012 pts
